# [HELOAY] 영화 리뷰 프로젝트

## 개발 환경

* Intellij IDEA Ultimate 2022.2.2
* Java 11
* Gradle 7.6
* Spring Boot 2.7.8

## 기술 세부 스택

Spring Boot

* Spring Web
* Spring Data JPA
* Thymeleaf
* Spring Security
* MariaDB
* Lombok

그 외
* HTML / CSS
* Bootstrap 5.2.2

* Spring Boot 계층형 아키텍처 기반 웹 어플리케이션 개발
* 스프링 프레임워크 기반 백엔드 구축
* JPA를 사용한  RDBMS 도메인 설계
* Spring Sercurity를 사용한 로그인
